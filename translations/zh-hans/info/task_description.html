在本任务中，将展示符号的组合。

预期行为：

  • 去掉开头的零，将只有零的数字转换为单个零；
  • 在数字之间的 +- 符号中，应取最后一个；
  • "=="表示重复上次操作；
  • "+="或"-="--在组合前对数字（或运算结果）进行加减运算（数字加倍/自身减倍）。
